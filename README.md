# Disappear Notes

A writing space that doesn't hold on.

Type anything. If you stop for 10 seconds, your words slowly dissolve —
fading into mist like they were never there. Save only what truly matters.

## The idea

Most notes apps are about keeping everything.
This one is about letting go.

## Features

- 10-second inactivity timer with a draining progress bar
- Text fades with a blur + mist animation (CSS only)
- Resume typing mid-fade — it snaps back
- "Keep this one" to permanently save a note
- Saved notes listed below with timestamps
- Nothing stored on a server. Nothing leaves your browser.

## Stack

- Vanilla HTML, CSS, JavaScript
- Cormorant Garamond + DM Mono (Google Fonts)
- Three files. No dependencies. No build step.

## Run it

Open `index.html` in any browser. That's it.

## Inspired by

The quiet anxiety of saving everything —
and what it might feel like to write without that weight.
